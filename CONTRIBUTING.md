# Contributing

This is a personal portfolio and is not currently accepting feature contributions. Evidence corrections and accessibility or security reports are welcome through a GitHub issue.

Before proposing a change, run `npm ci`, `npm run verify`, and `npm run test:e2e`. Keep project claims traceable to public repository evidence, preserve reduced-motion and static fallback behavior, and never add credentials or personal data. Do not replace the copyright status with an open-source license without the owner's explicit decision.
