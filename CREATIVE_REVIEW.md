# Creative review

The design avoids a template-like card wall. One node-and-path language changes character for rules, neural routing, streams, trajectories, pulses, and migrations. The homepage's scale and pacing create a portfolio journey, while the content remains ordinary HTML underneath.

The 3D scene is an optional enhancement triggered by visitor intent. This keeps the concept visible on mobile, reduced-motion, low-capability, and failed-WebGL paths without forcing the performance cost on recruiters.
