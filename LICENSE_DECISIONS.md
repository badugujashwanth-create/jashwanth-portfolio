# Portfolio license decisions

Reviewed 23 July 2026. This is an ownership and release decision record, not legal advice and not a grant of rights. No repository license was added by this review. Every listed working tree currently has no root `LICENSE` or `COPYING` file, so public visibility must not be interpreted as permission to reuse the code, media, datasets, or documentation.

Third-party package licenses remain governed by their packages and lockfiles regardless of the future license chosen for original repository code. Generated media, synthetic datasets, photographs, model artifacts, and upstream-derived code need separate provenance checks where noted.
