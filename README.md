# Jashwanth Portfolio

Source for Jashwanth Badugu's evidence-based engineering portfolio.

The site concept is **Systems in Motion**: an accessible WebGL system that maps real project themes—policy rules, local intelligence, workflow queues, migration state, and grounded data—onto one evolving visual language. A fast `/recruiter` route and semantic static content remain usable without WebGL.

Development is continuing on the `portfolio-launch` branch. Verified claims and exclusions are recorded in [portfolio-content-audit.md](portfolio-content-audit.md).
